# QUIZ com REACT
> O projeto conta com um jogo de perguntas e respostas interatito construído com React, com perguntas dentro da área de desenvolvimento de sistemas!

### Visão Geral

O projeto visa oferecer uma experiência de aprendizado e teste de conhecimento de forma divertida. Os usuários podem responder a uma série de perguntas e receber uma pontuação final.

-----

### Funcionalidades

  * **Perguntas de Múltipla Escolha:** Cada pergunta apresenta opções de resposta, onde apenas uma está correta.
  * **Controle de Progresso:** O usuário pode acompanhar em qual pergunta está (ex: 3 de 10).
  * **Pontuação Final:** Ao final do quiz, é exibida a pontuação total obtida.

-----

### Tecnologias Utilizadas

  * **React:** Biblioteca JavaScript para construção da interface do usuário.
  * **HTML/CSS/JavaScript:** Fundamentos da web para estruturação, estilização e lógica.
